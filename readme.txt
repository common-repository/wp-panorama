== Description ==
* View 360 Degree images in WordPress
* Thanks [Liviu Holhos](http://minimalisticstudio.com/) to write jQuery functions.

Features:
* Easy Installation.
* Using Mouse Wheel for move image.

== Installation ==
1. Upload `wp-panorama` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. To display Panoramic image, using function: `<?php wp_panorama('url_image', 'size_image', 'width', 'height'); ?>` in your theme.
For example: `<?php wp_panorama('http://upload.wikimedia.org/wikipedia/commons/c/cb/Persepolis-Panorama.jpg', 1400, 512, 250); ?>`
